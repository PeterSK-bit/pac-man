# Pac-Man

A classic Pac-Man game implemented in Java as a school project.

## Description

This is a Java implementation of the classic arcade game Pac-Man. The player controls Pac-Man through a maze, eating dots while avoiding ghosts. The goal is to eat all the dots in the maze to complete the level.

## Requirements

- Java 17 or higher
- IntelliJ IDEA (recommended)

## Running the Project

1. Clone the repository
   ```bash
   git clone https://github.com/PeterSK-bit/pac-man.git
   ```
2. Open the project in IntelliJ IDEA via **File > Open** and select the project folder
3. Let IntelliJ index the project
4. Run `Main.java` via the green play button or `Shift + F10`

## Controls

| Key | Action |
|-----|--------|
| `↑` or `w` | Move up |
| `↓` or `s` | Move down |
| `←` or `a` | Move left |
| `→` or `d` | Move right |
